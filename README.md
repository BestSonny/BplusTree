BPlusTree
=========

Implementation of B+ Tree (In-memory)

### Installation
1. Get the code.
  ```Shell
  git clone https://github.com/BestSonny/BplusTree.git
  cd BplusTree
  ```

2. Build and run the code. Either with Cmake or Make
  ```Shell
  make
  ./treesearch file_name
  ```

   or

  ```Shell
  mkdir build
  cd build
  cmake ../
  make
  ./treesearch file_name
  ```
